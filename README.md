# homebrew-proteus

Homebrew tap for [Proteus](https://github.com/GITDIDDY69/Proteus).

## Status

**No public release available yet.** This tap is reserved; no Cask
formula is currently published. Trying `brew install --cask proteus`
from this tap will fail until a Cask is added back when Proteus ships.

This README will be updated with install instructions when the first
public release is cut.
